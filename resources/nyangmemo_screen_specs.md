# 냥모 (NyangMemo) — 화면별 상세 설계서 v1.0

기준: 모바일 세로 (375×812pt 기준 / 목업 320×690). iOS·Android 공통, 플랫폼 차이는 각 화면에 명시.

핵심 기능 매핑
- **① 단축 동작 토글** → S2(빠른 캡처), S9(설정), 온보딩 S0-2
- **② LLM 자동 분류** → S3(분류 중), S4(분류 결과)
- **③ 카테고리별 GenUI** → S6(할 일), S7(참고자료), S8(아이디어)

---

## 정보 구조 (IA)

```
앱 진입
├─ (최초) 온보딩 S0-1 → S0-2 → S0-3
└─ 홈 S1
   ├─ FAB / 단축 동작 → 빠른 캡처 S2 → 분류 중 S3 → 분류 결과 S4
   ├─ 카테고리 카드 → GenUI 상세 (S6 할 일 / S7 참고자료 / S8 아이디어 …)
   ├─ 탐색 S5
   └─ 설정 S9
   (빈 상태 S10)
```

---

## S0. 온보딩 (3스텝)

**목적** 컨셉 전달 + 단축 동작 권한·설정 유도.

| 스텝 | 내용 | 핵심 요소 |
|---|---|---|
| S0-1 인사 | 모모 `wave` + "생각나는 걸 그냥 던져줘" | 마스코트, 시작하기 버튼, 인디케이터 ●○○ |
| S0-2 단축 동작 | iOS 뒷면 탭 / Android 전원 2회·퀵타일 안내 | 플랫폼별 안내 카드, "단축 동작 켜기" |
| S0-3 첫 메모 | 캡처 시트 미리보기 + 직접 입력 유도 | 캡처 시트, 예시 placeholder |

- **인터랙션:** 좌우 스와이프 또는 버튼으로 이동. 건너뛰기 상단 우측.
- **엣지케이스:** 단축 동작 권한 거부 → 나중에 설정에서 켤 수 있음 안내, 강제하지 않음.

---

## S1. 홈 (카테고리 대시보드)

**목적** 카테고리를 한눈에 보고, 가장 빠르게 새 메모를 시작.

**구성**
- 앱바: "오늘의 메모 🐾" + 검색 아이콘
- 모모 인사 배너(lavender): "모모가 어제 메모 4개를 정리했어요!" (어제 자동분류 요약)
- 카테고리 카드 리스트: 이모지 칩 + 이름 + 항목수/타입 + 타입 태그(TODO/REF/IDEA)
- FAB ＋ (peach, 우하단)
- 하단 탭: 홈·탐색·＋·모모·설정

**인터랙션**
- 카드 탭 → 해당 카테고리 GenUI 상세.
- 카드 길게 누르기 → 이름 변경/색 변경/삭제/병합.
- FAB 또는 단축 동작 → 빠른 캡처 S2.
- 당겨서 새로고침 → 분류 재정렬.

**엣지케이스** 카테고리 0개 → S10 빈 상태. 카드 많을 때 스크롤, 최근 활동순 정렬 기본.

---

## S2. 빠른 캡처 (바텀시트) — 핵심 기능 ①

**목적** 카테고리 고민 없이 **내용만** 즉시 입력. 단축 동작의 종착지.

**구성**
- 배경 화면은 살짝 어두워지고(딤), 바텀시트가 spring 슬라이드 업.
- 상단 그랩 핸들 + 모모 `think` + "그냥 떠오른 걸 적어요"
- 점선 입력 영역(자유 텍스트, 멀티라인, 자동 포커스 → 키보드 즉시 노출)
- 좌: 🎙️ 음성입력 / 📷 사진(OCR) · 우: "저장 ✓"(mint)

**인터랙션**
- 진입 즉시 커서 포커스 — 한 동작으로 바로 타이핑 가능(앱을 켜지 않아도 단축 동작으로 도달).
- 저장 → S3 분류 중. 저장 후 시트 닫히고 모모 반응.
- 아래로 스와이프 / 빈 곳 탭 → 취소(입력 내용 있으면 임시저장 확인).

**플랫폼**
- iOS: 설정 > 손쉬운 사용 > 뒷면 탭에 "냥모 열기" 단축어 연결. 잠금화면 위젯/컨트롤센터 토글 제공.
- Android: 빠른 설정 타일, 전원 버튼 2회(제조사 의존), 홈 화면 위젯, 어시스턴트 인텐트.

**엣지케이스** 빈 내용 저장 비활성. 오프라인 → 로컬 저장 후 온라인 시 분류(낙관적 UI). 매우 긴 텍스트 → 스크롤 입력.

---

## S3. 자동 분류 중 — 핵심 기능 ②

**목적** LLM이 분류를 찾는 동안의 대기 상태를 귀엽게.

**구성** 모모 `think`(대형) + 말풍선 "모모가 어울리는 분류를 찾고 있어요…" + 픽셀 프로그레스 `▮▮▮▯▯`.

**인터랙션** 보통 1–2초. 백그라운드 처리되며, 사용자는 기다리지 않고 다른 작업 가능(논블로킹 옵션). 완료 시 자동으로 S4.

**엣지케이스** LLM 응답 지연/실패 → 로컬 규칙(키워드)로 임시 분류 또는 "미분류"에 저장 후 재시도. 네트워크 없음 → 큐에 적재.

---

## S4. 분류 결과 — 핵심 기능 ②

**목적** 어디로 분류됐는지 보여주고, 사용자가 한 번에 수정 가능.

**구성** 모모 `love` + 결과 카드: "이 메모를 여기에 넣었어요" + 분류 pill("✅ 할 일") + 원문 인용 + [↺ 다른 분류로] / [확인 ✓].

**동작 (2)의 핵심)**
- 기존 카테고리와 의미가 맞으면 **매핑**, 없으면 **새 카테고리 생성**("새 분류 '여행' 만들었어요" 표시).
- 설정에서 "분류 전 확인" OFF면 이 화면을 건너뛰고 토스트만("할 일에 저장됨 · 변경").

**인터랙션** "다른 분류로" → 카테고리 선택 시트(직접 지정 시 LLM이 학습 신호로 반영). 확인 → 해당 GenUI로 이동 또는 홈 복귀.

**엣지케이스** 분류 모호 → 후보 2–3개 제시. 새 카테고리 남발 방지(유사도 임계값, 기존 우선).

---

## S5. 탐색 (검색)

**목적** 메모/카테고리 빠른 검색과 최근 항목.

**구성** 검색바 + 자주 찾는 분류 칩 + 최근 메모 리스트(분류·날짜 메타). 

**인터랙션** 입력 시 실시간 필터(제목·본문·분류·날짜). 의미 검색(임베딩) 옵션. 결과 탭 → 해당 메모.

**엣지케이스** 결과 없음 → 모모 `sleep` + "찾는 메모가 없어요". 

---

## S6. GenUI · 할 일 (체크리스트) — 핵심 기능 ③

**목적** "할 일" 성격 카테고리는 **체크리스트 UI**로 자동 구성.

**구성** 앱바(이모지+이름) + 태그 `GENUI · CHECKLIST` + 체크 아이템 리스트(체크박스+텍스트+추가시각) + 하단 모모 응원("2개 완료!").

**인터랙션** 체크박스 탭 → 완료(취소선+바운스). 완료 항목 하단 이동/접기. 스와이프 삭제. 항목 탭 → 편집. 정렬: 미완료 우선.

**GenUI 규칙** LLM이 메모를 액션 단위로 쪼개 체크 항목화(예: "우유, 달걀 사기" → 1 항목 또는 2 항목, 사용자 설정). 날짜 표현 감지 시 리마인더 제안.

---

## S7. GenUI · 참고자료 — 핵심 기능 ③

**목적** "참고자료/링크" 성격은 **저장일 기준 정렬 + 링크 요약** UI로.

**구성** 앱바 + 정렬 토글(⇅) + 날짜 헤더 그룹 + RefCard(제목 + `AI요약` 한두 줄 + 링크). 

**인터랙션** 카드 탭 → 원문/링크 열기. AI요약은 링크 메타·본문에서 LLM이 자동 생성. 정렬: 최신순 기본 ↔ 출처별. 

**GenUI 규칙** 메모에 URL 포함 시 og:title·요약 자동 추출. 텍스트만 있으면 핵심 1줄 요약. 중복 링크 병합.

**엣지케이스** 링크 죽음(404) → 흐리게 + "원본 없음". 요약 실패 → 원문 일부 표시.

---

## S8. GenUI · 아이디어 (타임라인) — 핵심 기능 ③

**목적** "아이디어/일기" 성격은 **시간순 타임라인 카드**로.

**구성** 앱바 + 태그 `GENUI · TIMELINE` + 좌측 컬러바가 있는 IdeaCard(제목+본문) 세로 나열.

**인터랙션** 카드 탭 → 펼쳐 전체 보기/편집. 관련 아이디어 묶기 제안. 

**GenUI 규칙** 카테고리의 "성격"을 LLM이 판단해 위 3가지(체크/레퍼런스/타임라인) 외에도 적절한 레이아웃 템플릿을 선택·조합. 새 카테고리 생성 시 기본 템플릿 자동 지정 후 사용 패턴으로 조정.

---

## S9. 설정 — 핵심 기능 ① 제어

**목적** 단축 동작·자동 분류·모모 옵션 관리.

**구성 (그룹별)**
- **빠른 입력:** 단축 동작 토글 / 잠금화면 위젯 / 음성 입력
- **자동 분류:** LLM 자동 분류 / 새 카테고리 자동 생성 / 분류 전 확인
- **모모:** 반응 애니메이션(동작 줄이기 연동)
- (기타) 데이터 백업/내보내기, 테마, 폰트 크기, 정보

**인터랙션** 토글 즉시 반영. 단축 동작 토글 → 플랫폼 설정 딥링크 안내.

---

## S10. 빈 상태 (Empty State)

**목적** 첫 사용/데이터 없음을 따뜻하게.

**구성** 모모 `sleep`(대형) + "아직 메모가 없어요" + "＋ 버튼을 누르거나 버튼 두 번 눌러 첫 메모를 남겨봐요" + FAB 강조.

**변형** 카테고리 내부 비었을 때, 검색 결과 없을 때 각각 문구·표정 변주.

---

## 공통 규칙

- **하단 탭(5):** 홈 · 탐색 · ＋(중앙, 캡처) · 모모(상태/통계) · 설정. 활성 탭 peach.
- **상태바:** 시스템 시간/배터리 영역 존중, 노치 대응 세이프영역 패딩.
- **에러/토스트:** 페이퍼 카드 + ink 텍스트, 짧게. 파괴적 동작은 확인 다이얼로그.
- **로딩:** 모모 + 픽셀 프로그레스로 통일. 스피너 대신 도트 모션.
- **반응형:** 작은 기기 1열 카드, 큰 기기/가로 2열 그리드 허용.
