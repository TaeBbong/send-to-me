# 냥모 (NyangMemo) — 디자인 시스템 v1.0

> 생각나면 톡 던지고, 정리는 고양이 마스코트 "모모"와 LLM에게 맡기는 자동 분류 메모 앱.
> 컨셉: **Pastel Cream + 도트(픽셀) 고양이 + 손그림 보더** — 셋로그 / habit puzzle / 고양이 책집사 계열의 심플하고 귀여운 톤.

---

## 1. 브랜드 기초

| 항목 | 값 |
|---|---|
| 앱 이름 | 냥모 (NyangMemo) |
| 마스코트 | 도트 고양이 **모모(Momo)** |
| 톤 & 무드 | 따뜻함, 가벼움, 귀여움, 부담 없음 |
| 키워드 | 던지기(toss) · 알아서(auto) · 귀여움(cozy) |
| 보이스 | 친구 같은 반말, 응원하는 말투 ("내가 정리할게!", "잘하고 있어요") |

---

## 2. 컬러 토큰

순수 블랙·화이트를 쓰지 않습니다. 배경은 크림, 텍스트는 따뜻한 브라운(ink)을 사용해 종이 같은 느낌을 만듭니다.

### 2.1 코어 (Neutral)

| 토큰 | HEX | 용도 |
|---|---|---|
| `--cream` | `#FBF4E6` | 앱 기본 배경 |
| `--cream-deep` | `#F3E8D2` | 섹션 구분 배경 |
| `--paper` | `#FFFDF8` | 카드 / 시트 표면 |
| `--ink` | `#5A4A3A` | 기본 텍스트 · 보더 |
| `--ink-soft` | `#8C7B66` | 보조 텍스트 · 메타 |
| `--line` | `#E2D4BC` | 연한 구분선 · 도트 그리드 |
| `--line-strong` | `#C9B79A` | 보더(약) · 비활성 |

### 2.2 포인트 (Pastel Accent)

카테고리마다 아래 포인트 컬러 1개를 자동 배정합니다.

| 토큰 | HEX | 의미 / 매핑 |
|---|---|---|
| `--peach` / `--peach-deep` | `#F4A988` / `#E8876B` | **주요 액션**, FAB, 강조 |
| `--mint` / `--mint-deep` | `#9FD8BE` / `#6FBF9B` | **완료 / 저장**, 할 일 카테고리 |
| `--lav` / `--lav-deep` | `#C7B5E8` / `#A98FD6` | **AI / 자동 분류** 관련 |
| `--butter` | `#F6D67E` | 태그 · 아이디어 카테고리 |
| `--sky` | `#A6CCE8` | 참고자료 카테고리 |
| `--rose` | `#F2A7B5` | 볼터치 · 장보기 카테고리 |

### 2.3 컬러 규칙

- 텍스트/배경 대비는 WCAG AA(4.5:1) 이상 — 본문 텍스트는 항상 `--ink` 사용. 파스텔 위에 흰 텍스트를 쓰지 않습니다(대비 부족). 버튼 위 텍스트는 흰색 대신 `--ink` 또는 짙은 동계색.
- 포인트 컬러는 면적의 20% 이하로. 크림/페이퍼가 화면을 지배합니다.
- 카테고리 색 자동 배정 순서: peach → mint → sky → butter → rose → lav → (반복, 명도 변주).

| 역할 | 폰트 | 크기(px) | 비고 |
|---|---|---|---|

> 한글 대체: Jua↔여기어때 잘난체/배민 주아, Gaegu↔나눔손글씨, Press Start 2P↔갈무리(Galmuri)/둥근모. 앱 빌드 시 라이선스 확인.

## 4. 간격 · 모서리 · 그림자

- **Spacing (8pt grid):** 4 · 8 · 12 · 16 · 24 · 32px.
- **Radius:** 카드 18px, 시트/모달 22–28px, 작은 칩/버튼 14px, 토글/픽셀 요소 6–7px.
- **Border:** 손그림 느낌의 두꺼운 보더 `2–3px solid var(--ink)`. 이것이 이 디자인의 핵심 시그니처입니다.
- **Shadow (오프셋, 블러 없음):** `--shadow: 4px 4px 0 rgba(90,74,58,.14)` / `--shadow-sm: 2px 2px 0 …`. 종이를 겹친 듯한 스탬프 그림자.
- **배경 텍스처:** 18px 간격 도트 그리드(`radial-gradient`)로 모눈종이 느낌.

---

## 5. 마스코트 "모모"

16×16 픽셀 도트 고양이. 상황별 표정으로 감정을 전달하며, 빈 화면·로딩·완료 등 빈 순간을 채웁니다.

| 무드 | 상황 | 표현 |
|---|---|---|
| `wave` | 온보딩 인사 | 손 흔들기 |
| `happy` | 홈, 완료 응원 | 웃는 눈 |
| `think` | 자동 분류 중 | 골똘히 + 물음표 반짝 |
| `love` | 저장 완료 | 하트 눈 |
| `sleep` | 빈 화면 | 감은 눈, 쿨쿨 |

규칙: 한 화면에 모모는 1마리만. 픽셀 렌더링 유지(`image-rendering:pixelated`). 과한 등장 금지 — 의미 있는 순간에만.

## 6. 컴포넌트 라이브러리

| 컴포넌트 | 설명 | 상태 |
|---|---|---|
| **CategoryCard** | 이모지 칩 + 이름 + 개수 + 타입 태그 | default / pressed |
| **CheckItem** | 체크박스 + 텍스트 + 시각, 완료 시 취소선 | unchecked / checked |
| **RefCard** | 제목 + AI요약 + 링크, 날짜 헤더로 그룹 | default |
| **IdeaCard** | 좌측 컬러바 + 제목 + 본문 (타임라인) | default |
| **CaptureSheet** | 바텀시트, 점선 입력 영역, 음성/사진 + 저장 | empty / typing |
| **ResultCard** | 분류 결과 pill + 원문 + 확인/재분류 | — |
| **FAB** | 우하단 ＋, peach, 두꺼운 보더 | default / pressed |
| **BottomNav** | 홈·탐색·＋·모모·설정 (5칸) | active 강조(peach) |
| **Toggle** | 둥근 스위치, on=mint / off=line | on / off |
| **Chip / Tag** | 칩(둥근) · 태그(픽셀 라벨) | — |
| **Pill** | 분류명 표시 (lavender) | — |

상호작용: 누름 시 `scale(.96)` + 그림자 축소(눌리는 느낌). 트랜지션 80–160ms, ease-out.

---

## 7. 모션

- **입력창 토글:** 바텀시트가 아래에서 spring으로 슬라이드 업(220ms). 단축 동작 트리거 시 동일.
- **자동 분류:** 모모 think → 픽셀 프로그레스(▮▮▮▯▯) → 결과 카드 pop-in(scale 0.9→1).
- **체크 완료:** 체크박스 mint 채워짐 + 텍스트 취소선 + 살짝 바운스.
- 모든 모션은 짧고 통통 튀게(playful), 과하지 않게.

---

## 8. 접근성

- 본문 텍스트 대비 4.5:1 이상(ink on cream/paper 충족).
- 터치 타깃 최소 44×44pt (FAB·아이콘버튼·토글 준수).
- 색만으로 정보 전달 금지 — 카테고리는 색 + 이모지 + 텍스트 라벨 병행.
- 폰트 크기는 시스템 다이내믹 타입에 대응(상대 단위 권장).
- 모모 애니메이션은 "동작 줄이기(reduce motion)" 설정 시 정적 이미지로 대체.
